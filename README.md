Engenharia de Prompt e Aplicações de IA
📌 Sobre a Disciplina

A disciplina Engenharia de Prompt e Aplicações de Inteligência Artificial tem como objetivo apresentar os conceitos, técnicas e boas práticas para interagir de forma eficiente com modelos de Inteligência Artificial, especialmente modelos de linguagem (LLMs).
Os alunos aprendem a estruturar prompts, criar aplicações baseadas em IA e integrar essas tecnologias em soluções reais.

🎯 Objetivos

Compreender os fundamentos de Inteligência Artificial Generativa.

Aprender técnicas de engenharia de prompt para melhorar respostas de modelos de IA.

Desenvolver aplicações práticas utilizando APIs de IA.

Explorar automação, análise de dados e geração de conteúdo com IA.

Discutir ética, segurança e limitações da IA.
